Создать класс и 2 наследника.
Создаем класс Person с одноименными свойствами - name, email , phone.



Создаем класс метод showInfo который возвращает строку вида:

"Имя: Иван Иванов. Имейл: ivanivanov@gmail.com. Телефон: 123-213-12."



Создаем два дочерних класса: Coach (Преподаватель) и Student



Для класса Student добавляем дополнительное свойство rating которое хранит средний балл студента



Для преподавателя добавляем новое свойство course которое будет хранить название курса который ведет преподаватель (например HTML, PHP, т.п).

Например:

"Имя: Иван Иванов. Имейл: ivanivanov@gmail.com. Телефон: 123-213-12. Курс: PHP"



Для каждого класса переопределяем метод showInfo и дополняем его выводом нового свойства.



Создаем массив(10 элементов) содержащий массивы с учетными данными студентов и преподавателей:



[

'name' => 'John Doe',

'email' => 'john@gmail.com',

'phone' =>'123-213-12',

'role' => 'student', // или 'coach'

'rating' => '95', // для студента

'course' => 'PHP', // для преподавателя

]



Создаем новый пустой массив и в цикле проверяем значение элемента с ключем 'role' и в зависимости от его значения создаем новый объект либо класса Coach либо Student и добавляем в наш новый массив.



В HTML с помощью элементов <ul><li>...</li></ul> выводим через цикл информацию вызовом метода showInfo()



Не забываем объявлять конструкторы и создавать объекты через них. А также выносить классы в отдельные файлы.



Дополнительно по возможности стараемся пользоваться модификаторами доступа:

public - для свойств и методов которые вы будете вызывать в программе

private - для свойств которые будут использоваться только внутри методов текущего класса

protected - только для методов которые переопределяются в дочернем классе
